# Резидентная программа для мониторинга значений регистров

## Установка
Для установки резидентной программы необходимо запустить RESID.com

### Аргументы командной строки при установке
При установке программы можно изменить вид рамки при помощи аргументов:\
- /no  - отключает рамку
- /f=symb - отображает рамку из одинаковых символов "symb"

## Использование
После установки при каждом нажатии RShift в центре экрана будет появляться табличка, содержащая значения 4 основных регистров,
4 индексных регистров, 4 сегментных регистров, а также IP (Instruction Pointer)
